Image tag

  • img tag is used to display image on the web page
  • img tag is an empty tag that contains attributes only, closing tags are not used in image element
  • the <img> tag is used to embed an image in an HTML page
Note: Images are not technically inserted into a web page; images are linked to web pages.
The <img> tag has two required attributes:
  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image,if the image for some reason cannot be displayed
